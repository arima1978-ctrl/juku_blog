# アンイングリッシュグループ 教室ネットワーク(参考情報)

`config/juku.yaml` は**小幡校(米澤由里子塾長)専用の設定**。他教室の情報は、area/author等の
フィールドに混ぜて使わないこと(生徒の通学区域や塾長ペルソナが教室ごとに異なるため)。
将来的に他教室向けにこのシステムを展開する場合は、教室ごとに別の `config/juku.yaml` を用意する。

## 共通事項(グループ全体で提供)
どちらの教室も同じコースラインナップを提供しており、コース名・ブランド名はグループ全体で統一されている:
アンイングリッシュクラブ(英会話)/アンそろばんクラブ/アン美文字クラブ(習字)/アンプログラミングクラブ/
アン将棋クラブ/アンさんこくキッズ/アン進学ジム(個別指導)/アンまなびワールド。
対象学年は各教室とも年少〜高校生。
