## 基本情報
- アンイングリッシュグループ傘下の個別指導塾。37年の歴史、累計10,000名以上が利用
- 愛知県・岐阜県に70校舎超のネットワーク
- 対象学年: 小学部(小1〜小6)、中学部(中1〜中3)、高校部(高1〜高3)

## 指導方針
- 「解き方のトレーニング塾」というコンセプト
- 生徒個々に合わせた学習計画、「勉強する力」の育成を重視
- 教育コンサルタント小田原健一氏のメソッドが基礎

## 授業形式
- 映像授業(全授業をタブレット収録、自宅学習にも対応)
- 個別指導(講師1名が生徒10〜15名を担当)
- 週1回〜フリー通塾(バイキング方式: 教室開講日・開講時間ならいつでも通塾可、追加費用なし)

## 小学部の強み
1. 教科書完全準拠教材による中学基礎構築
2. 入塾時にタブレット配布、自宅でも塾と同等の学習環境
3. 学期ごとの「アン小学生テスト」結果をもとにした苦手分析・個別カリキュラム調整
4. アンイングリッシュクラブと連携した英語4技能(聞く・話す・読む・書く)強化
5. 映像授業後のWeb確認テスト(理解度チェック)
6. アンそろばんクラブとの併用による計算力強化

## 中学部の強み
1. バイキング方式(通塾自由・追加費用なし)
2. 現状学力・自立度を考慮した個別メソッド指導
3. 「決められた内容が終わった時が勉強の終わり」という習慣づけ
4. 各中学の日程に合わせた定期テスト対策勉強会・「+10点スペシャル特訓」
5. 70校舎ネットワーク・長年のテストデータに基づく進路指導
6. マーカーでラインを引きながら解説する映像授業設計

## 学習の流れ(5ステップ)
カウンセリング → 映像授業受講 → マンツーマン指導(理解度チェック) → 問題演習(定着まで反復) → 振り返り・宿題確認

## 使用教材
- 教科書準拠教材(小学部)
- EDUPlus(名大SKY開発の映像教材。全国50,000名以上が利用)
- タブレット(入塾時配布)

## イベント・サポート
アン小学生テスト/アン中学生テスト(年3回)、春夏冬の講習会、定期テスト対策勉強会、全県模試、年2回個別面談、進学情報説明会 等

## 小幡校独自の取り組み(実データ、米澤塾長のLINE連絡より確認・2026-07-07)
- チェックテストで満点を一定回数(4回)積み重ねると、ちょっとしたお菓子のプレゼントがもらえる仕組みがある。もともと個別指導(進学ジム)の小学4年生向けだったが、小学3年生以下の「さんこくキッズ」受講生にも対象を拡大した
- 記事で触れる場合は「教室独自の取り組みとして」等の書き方にとどめ、対象学年・回数などの細部は変動しうる前提で断定しすぎない
